自Vim 8.0起，只要包含channel和job特性，Vim即可以支持异步（asynchronous IO）,Start another process through the job,Process communication through channels. You can make the plugin or script have a better  experience by using Vim's asynchronous support,For example, asynchronous  compilation prevents Vim from blocking during compilation,Vim gets  faster speeds when asynchronous syntax highlighting comes.
